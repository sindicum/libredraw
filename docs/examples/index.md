# Live Demo

Try LibreDraw directly in your browser. Use the toolbar buttons on the map to draw and edit polygons.

**Controls:**
- **Draw** (pencil icon): Click to add vertices, double-click to close
- **Select** (cursor icon): Click a polygon to select, drag vertices to edit
- **Delete** (trash icon): Delete the selected polygon
- **Undo / Redo**: Undo or redo the last action
